# Lovable Project Starter Kit

One-click professional foundation for any Lovable.dev project.

## How to use
1. Clone or open this repo
2. Copy the prompt you need from `/prompts/`
3. Paste into a new Lovable project
4. Turn on Project Knowledge and add the outputs

## Files
- `prompts/` → all 8 templates
- `workspace-knowledge.md` → paste into your Lovable Workspace Knowledge (set once)
- `project-checklist.md` → run every new project

Made for speed. Updated March 2026.
